  • Featured project

    llm-agent-control-plane

    A defensive reference implementation for tool-connected LLM agents. Agents may propose actions; an external control plane authorizes them.

    # Agent proposes; control plane authorizes
proposal = agent.plan(user_input)
decision = control_plane.authorize(proposal, policy, identity)
if decision.allowed:
    executor.run(proposal.action)

    Source: policy_engine.py in llm-agent-control-plane

  • Research note · June 2026

    Detection Engineering Note: OAuth Anomaly Baseline

    Baseline OAuth grant volume and client diversity per user, then alert on statistically unusual token activity. Useful for spotting consent phishing, token theft, and misconfigured integrations in authorized environments.

    index=authentication sourcetype=oauth:token
| stats dc(client_id) AS clients dc(scope) AS scopes count BY user
| eventstats avg(count) AS avg_grant stdev(count) AS stdev_grant BY user
| eval threshold=avg_grant+(3*stdev_grant)
| where count > threshold AND clients > 1
